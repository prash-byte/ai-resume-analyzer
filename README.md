# Resume Analyzer

A Python app that analyzes a resume against a job description using AI.

## Features
- Match score
- Strengths & weaknesses
- Improvement suggestions

## Setup

1. Install:
pip install -r requirements.txt

2. Add API key in `.env`:
OPENAI_API_KEY=your-key

3. Run:
python app.py

**Example Output:**

Match Score: 75/100

Strengths:
- Strong Python and ML skills
- Relevant project experience

Missing Skills:
- Deep Learning
- SQL

Suggestions:
- Add SQL projects
- Highlight problem-solving experience
